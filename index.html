SHINRA

System Hardware Info for R and Analytics

View SHINRA on GitHub

SHINRA is a collection of packages, or meta-package, for R for accessing and understanding various system hardware resources.

  • memuse is a collection of utilities for managing memory information (as multiples of bytes), and accessing system hardware values, such as the amount of system ram.

  • merkhet is a package for times. With it you can query system uptimes, process runtimes, and more.

  • okcpuid is a simple set of utilities for understanding the system CPU.

  • scribe is a set of enhancements and ease-of-use features for the pbdPAPI package. This adds benchmarking and visualization tools for hardware performance counter data.
